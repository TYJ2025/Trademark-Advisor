# Trademark Advisor

Senior Trademark Strategist & Portfolio Advisor — an interactive single-file HTML tool for trademark portfolio analysis.

## Features
- Bilingual (English / Traditional Chinese) analysis reports
- 5-step SOP: Status Audit, Risk Assessment, Strategic Expansion, Enforcement, Documentation
- Actual Use vs. Registration comparison analysis
- Nice Classification coverage and gap analysis
- JSON portfolio import support
